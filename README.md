# traffic-rl

What's the best red/green schedule for traffic lights — minimum total waiting, kept
realistic? From Webster's 1958 formula to multi-agent RL on one question.

Stages: metrics & realism constraints → simulator → classical baselines (fixed-time,
Webster, actuated, max-pressure) → RL single intersection → RL network (does a green
wave emerge?) → writeup. Status: stage 0.
